Descobrindo tags

<button></button>

A tag <button> define um botão clicável.
Dentro de um elemento <button>, você pode colocar texto (e tags como <i>, <strong>, <br>, <img> etc.).
Isso não é possível com um botão criado com o elemento <input>!
Dica: Sempre especifique o tipo do atributo para um elemento <button>, para informar aos navegadores que tipo de botão é esse.
Dica: Você pode facilmente estilizar botões com CSS!

<cite></cite>

A tag <cite> define o título de um obra de arte (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura etc.).
Nota: O nome de uma pessoa não é o título de uma obra.
O texto no elemento <cite> geralmente é formatado em itálico.

<details></details>

A tag <details> especifica detalhes adicionais que o usuário pode abrir e fechar quando quiser.
A tag <details> é frequentemente usada para criar um widget interativo que o usuário pode abrir e fechar. Por padrão, o widget está fechado. Quando aberto, ele se expande e exibe o conteúdo dentro.
Qualquer tipo de conteúdo pode ser colocado dentro da tag <details>.
Dica: A tag <summary> é usada em conjunto com <details> para especificar um cabeçalho visível para os detalhes.

Exemplo de utilização:

Epcot Center   

O Epcot é um parque temático do Walt Disney World Resort com atrações emocionantes, pavilhões internacionais, fogos de artifício premiados e eventos especiais da estação.

Nota: O elemento details não é suportado no Edge (versão anterior 79).

<mark></mark>

A tag <mark> define o texto marcado. Use a tag <mark> se desejar destacar partes do seu texto.

Exemplo de utilização: Vou colocar a palavra "amor" em detaque: amor.
