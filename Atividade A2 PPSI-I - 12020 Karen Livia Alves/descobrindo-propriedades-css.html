Descobrindo propriedades CSS

Outline

A propriedade CSS outline é uma propriedade abreviada para configurar uma ou mais das propriedades de contorno outline-style, outline-width e outline-color em uma única declaração. Na maioria dos casos o uso abreviado é preferível e mais conveniente. Clique aqui para saber mais.

Exemplo de utilização:


Código Visualização
#exOutline{
        outline: 1px solid black;
}

Exemplo outline

Text-decoration

A propriedade CSS text-decoration é usada para definir a formatação de underline, overline, line-through ou blink. As decorações underline e overline são posicionadas abaixo e acima do texto (respectivamente), e line-through cortando-o. Clique aqui para saber mais.

Exemplo de utilização:


Código Visualização
#exTextDecoration{
        text-decoration: line-through;
}

Exemplo text-decoration

Opacity

A propriedade CSS opacity especifica a transparência de um elemento, isto é, o grau no qual o background atrás do elemento é sobreposto. Clique aqui para saber mais.

Exemplo de utilização:


Código Visualização
#exOpacity{
        opacity: 0.6;
}

Exemplo opacity

Text-shadow

A propriedade CSS text-shadow acrescenta sombras ao texto. Ela aceita uma lista de sombras separadas por vírgula que serão aplicados ao texto e ao text-decorations do elemento.
Cada sombra é especificada como um deslocamento do texto, juntamente com valores opcionais de cor e raio de desfoque. Clique aqui para saber mais.

Exemplo de utilização:


Código Visualização
#exText-shadow{
        text-shadow: 2px 2px purple;
}

Exemplo text-shadow
